# Agentic Results Viewer

Interactive Streamlit-based viewer for analyzing experiment results from the OpenDsStar framework.

## Features

### Single Results Viewer (`agentic_results_viewer.py`)
- Load and visualize results from `*_output.json` files
- Filter by score range and source file
- View aggregated metrics (avg score, steps, tokens)
- Inspect individual question trajectories
- View error analysis results

### Comparison Viewer (`agentic_results_comparisson.py`)
- Compare two sets of results side-by-side
- Identify common questions and unique questions
- Filter by score ranges and delta
- View comparative metrics
- Inspect trajectories from both sources

## Installation

Ensure streamlit is installed:
```bash
pip install streamlit pandas
```

## Usage

### Single Results Viewer

```bash
# Using streamlit directly
streamlit run src/ui/agentic_results_viewer/agentic_results_viewer.py

# Or from virtual environment
.venv/bin/streamlit run src/ui/agentic_results_viewer/agentic_results_viewer.py
```

Then in the sidebar:
1. Enter a path to a single `*_output.json` file, OR
2. Enter a directory path to load all `*_output.json` files recursively

### Comparison Viewer

```bash
# Using streamlit directly
streamlit run src/ui/agentic_results_viewer/agentic_results_comparisson.py

# Or from virtual environment
.venv/bin/streamlit run src/ui/agentic_results_viewer/agentic_results_comparisson.py
```

Then in the sidebar:
1. Enter Path A (baseline) - file or directory
2. Enter Path B (comparison) - file or directory

## File Format

The viewer expects `*_output.json` files generated by the experiment pipeline with the following structure:

```json
{
  "run_id": "experiment_name",
  "timestamp": "2026-02-12T10:52:22.460879",
  "agent_type": "ds_star",
  "summary": {
    "total_results": 3,
    "avg_score": 0.93,
    "passed_count": 3,
    "passed_rate": 1.0
  },
  "items": [
    {
      "question_id": "q1",
      "output": {
        "question_id": "q1",
        "answer": "The answer...",
        "metadata": {
          "question": "What is...?",
          "trajectory": [...],
          "input_tokens": 1000,
          "output_tokens": 200,
          "ground_truth": {
            "answers": ["correct answer"]
          }
        }
      },
      "evaluations": [
        {
          "question_id": "q1",
          "score": 0.9,
          "passed": true,
          "details": {...}
        }
      ]
    }
  ]
}
```

## Features by Tab

### Aggregation Tab
- Overview metrics (count, avg score, avg steps, avg tokens)
- Sortable table with all questions
- Progress bars for scores
- Multi-file source tracking

### Detailed Inspection Tab
- Question selector with score preview
- Full question and answer display
- Ground truth comparison
- Token usage metrics
- Error analysis (if available)
- Complete trajectory visualization

### Trajectory Rendering
- **DS-Star format**: Structured steps with plan/code/execute/verify
- **String format**: CodeAct/ReAct style with actor identification
- Expandable sections for each step
- Syntax highlighting for code
- JSON formatting for structured data

## Tips

1. **Loading multiple files**: Point to a directory containing multiple `*_output.json` files to compare across experiments
2. **Filtering**: Use score sliders and file filters to focus on specific subsets
3. **Comparison**: Use delta filters to find questions where performance changed significantly
4. **Trajectory inspection**: Expand trajectory steps to debug agent behavior

## Troubleshooting

**Import errors**: Make sure you're running from the project root and the virtual environment is activated:
```bash
cd /path/to/OpenDsStar
source .venv/bin/activate
streamlit run src/ui/agentic_results_viewer/agentic_results_viewer.py
```

**No data loaded**: Verify the path points to valid `*_output.json` files generated by the experiment pipeline.
